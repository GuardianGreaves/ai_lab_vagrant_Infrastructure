Проект создан в рамках выполнения задания по развертыванию инфраструктуры для AI-агента.

На текущем этапе реализована базовая инфраструктура с автоматическим созданием виртуальных машин и установкой необходимых сервисов через Vagrant Shell Provisioning.

## Реализовано

### Автоматическое создание виртуальных машин

Через Vagrant создаются 4 виртуальные машины:

| VM | IP | Назначение |
|---|---|---|
| postgres | 192.168.56.10 | PostgreSQL сервер |
| n8n | 192.168.56.11 | Automation platform |
| ollama | 192.168.56.12 | Local LLM сервер |
| wiki | 192.168.56.13 | Wiki.js документация |

Используется:
- Ubuntu 22.04 LTS
- VirtualBox provider
- Vagrant Shell Provisioning


## PostgreSQL VM

Автоматически выполняется:

- установка PostgreSQL
- запуск и добавление в автозагрузку systemd
- настройка сетевого доступа
- разрешение подключения только из внутренней сети Vagrant
- создание пользователей и баз данных

Созданные базы:


n8n
wikijs
agent_memory


Созданные пользователи:


n8n_user
wiki_user
agent_user



## n8n VM

Автоматически выполняется:

- установка Docker
- запуск контейнера n8nio/n8n
- подключение к PostgreSQL
- настройка хранения данных в PostgreSQL

n8n доступен через:


http://192.168.56.11:5678


На текущем этапе:
- UI доступен
- контейнер запускается автоматически
- база данных вынесена в отдельную VM


## Ollama VM

Автоматически выполняется:

- установка Docker
- запуск контейнера Ollama
- создание Docker volume для хранения моделей
- загрузка LLM модели из библиотеки Ollama

Текущая модель:


mistral:latest


API доступен:


http://192.168.56.12:11434



Проверка:

```bash
curl http://192.168.56.12:11434/api/tags

Автоматизация

Все базовые действия выполняются автоматически через Vagrant provisioning:

vagrant up

Команда создает виртуальные машины и выполняет установку сервисов.
