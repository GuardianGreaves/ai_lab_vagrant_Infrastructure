# AI Agent Infrastructure Lab

Проект создан в рамках выполнения задания по развертыванию инфраструктуры для AI-агента.

На текущем этапе реализована базовая инфраструктура с автоматическим созданием виртуальных машин и установкой необходимых сервисов через **Vagrant Shell Provisioning**.

---

# Реализовано

## Автоматическое создание виртуальных машин

Через Vagrant создаются 4 виртуальные машины:

| VM | IP-адрес | Назначение |
|---|---|---|
| postgres | `192.168.56.10` | PostgreSQL сервер |
| n8n | `192.168.56.11` | Automation platform |
| ollama | `192.168.56.12` | Local LLM сервер |
| wiki | `192.168.56.13` | Wiki.js документация |

Используется:

- Ubuntu 22.04 LTS
- VirtualBox provider
- Vagrant Shell Provisioning

---

# PostgreSQL VM

Автоматически выполняется:

- установка PostgreSQL
- запуск сервиса через systemd
- добавление PostgreSQL в автозагрузку
- настройка сетевого доступа
- разрешение подключений только из внутренней сети Vagrant
- создание пользователей и баз данных

## Созданные базы данных

```text
n8n
wikijs
agent_memory
```

## Созданные пользователи

```text

wiki_user
agent_user
```

- установка Docker
- запуск контейнера `n8nio/n8n`

- настройка хранения данных n8n в PostgreSQL

# Автоматизация

Все базовые действия выполняются автоматически через Vagrant provisioning.

Запуск инфраструктуры:

```bash
vagrant up
```

Команда выполняет:

1. Создание виртуальных машин
2. Настройку внутренней сети
3. Запуск provisioning-скриптов
4. Установку необходимых сервисов

# Цель проекта

Получить полностью воспроизводимую инфраструктуру AI-агента, где:

- **n8n** отвечает за автоматизацию workflow
- **Ollama** предоставляет локальную LLM
- **PostgreSQL** хранит данные приложений и память AI-агента
- **Wiki.js** используется как база технической документации
