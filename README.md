# AI Lab - Vagrant Infrastructure

Домашний DevOps стенд для изучения Linux, Docker, PostgreSQL, автоматизации и локальных AI систем.

Проект был создан в рамках выполнения практического задания 

Все виртуальные машины разворачиваются локально через **Vagrant + VirtualBox**, настройка сервисов выполняется автоматически через shell provisioning.

## Выполненное задание

### 1. Виртуальная инфраструктура

Развернуты 4 Linux виртуальные машины:

* PostgreSQL
* n8n
* Ollama
* Wiki.js

Все машины находятся в одной приватной сети и имеют статические IP адреса.

### 2. PostgreSQL сервер

Настроен отдельный сервер PostgreSQL:

* включен сетевой доступ только из подсети виртуальных машин
* создан пользователь и база для n8n
* создан пользователь и база для Wiki.js
* создан пользователь и база для памяти AI агента

Базы данных:

```
n8n
wikijs
agent_memory
```

### 3. n8n Automation Platform

На отдельной VM:

* установлен Docker
* запущен n8n контейнер
* настроено хранение данных через PostgreSQL
* проверено сохранение workflow и credentials после перезапуска контейнера

Web интерфейс:

```
http://192.168.56.11:5678
```

### 4. Ollama Local AI

На отдельной VM:

* установлен Docker
* запущен Ollama сервер
* установлена LLM модель из библиотеки Ollama
* проверена генерация через CLI и API

API:

```
http://192.168.56.12:11434
```

Используемая модель:

```
mistral:latest
```

### 5. AI Agent workflow

В n8n создан workflow:

```
Chat Message
      |
      v
AI Agent
      |
      v
Ollama Chat Model
```

Проверено:

* отправка сообщений в Ollama
* получение ответов модели
* обработка нескольких различных запросов

### 6. Память AI агента

Настроено хранение памяти AI агента в PostgreSQL.

Проверенный сценарий:

```
Пользователь: Меня зовут Саня

Пользователь: Как меня зовут?

Restart n8n

Пользователь: Как меня зовут?
```

После перезапуска n8n контекст сохраняется.

### 7. Wiki.js Knowledge Base

На отдельной VM:

* установлен Docker
* запущен Wiki.js
* подключен PostgreSQL backend
* создана документация инфраструктуры

Созданные страницы:

* Архитектура AI-агента
* Описание схемы инфраструктуры

В документации описаны:

* компоненты системы
* связи между сервисами
* сетевые порты
* расположение данных

### 8. Автоматизация развертывания

Весь стенд разворачивается автоматически через:

* Vagrantfile
* Shell provisioning scripts
* Environment variables

После выполнения:

```bash
vagrant up
```

создаются и настраиваются все виртуальные машины.
